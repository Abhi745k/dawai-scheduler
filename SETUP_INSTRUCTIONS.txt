=============================================================================

This project contains a medicine reminder agent for Indian families.
Two versions are provided:

1. Kaggle Notebook Demo (medicine_reminder_agent.ipynb)
2. Mobile App Prototype (main.py)

=============================================================================
üìã REQUIREMENTS
=============================================================================

- Python 3.8 or higher
- pip package manager

=============================================================================
üöÄ QUICK START FOR KAGGLE DEMO
=============================================================================

1. Install Required Packages:
   
   pip install -r requirements.txt

2. Open Kaggle Notebook:
   
   - Upload medicine_reminder_agent.ipynb to Kaggle
   - Or open it in Jupyter Notebook locally:
     jupyter notebook medicine_reminder_agent.ipynb

3. Run All Cells:
   
   - Click "Run All" or execute cells sequentially
   - The notebook demonstrates all features interactively

=============================================================================

üîä TEXT-TO-SPEECH NOTES
=============================================================================

- Uses Google Text-to-Speech (gTTS) library
- Requires internet connection for audio generation
- Audio files saved as .mp3 format
- Supports Hindi and English languages

For offline TTS, use pyttsx3 (included in requirements):
  import pyttsx3
  engine = pyttsx3.init()
  engine.say("Your message")
  engine.runAndWait()

üõ†Ô∏è TROUBLESHOOTING
=============================================================================

Issue: gTTS not working
Solution: 
  - Check internet connection
  - Install/reinstall: pip install --upgrade gtts
  - Alternative: Use pyttsx3 for offline TTS

Issue: Kivy not installing
Solution:
  - Follow official guide: https://kivy.org/doc/stable/gettingstarted/installation.html
  - For Windows: pip install kivy[base] kivy_examples

Issue: Audio not playing in notebook
Solution:
  - Audio playback depends on browser support
  - Download .mp3 files and play externally
  - Use mobile app for automatic playback
